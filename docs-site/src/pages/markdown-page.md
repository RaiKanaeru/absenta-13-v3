# Ringkasan Sistem absenta13

Halaman ini berisi ringkasan cepat sistem absenta13 untuk pembaca teknis.

## Fokus Utama

- Absensi siswa dan guru berbasis jadwal.
- Manajemen data sekolah (kelas, mapel, guru, siswa).
- Laporan dan export Excel dengan kop laporan.
- Monitoring, backup, dan operasional sistem.

## Komponen Teknis

- Frontend: React + TypeScript + Vite.
- Backend: Node.js + Express.
- Database: MySQL.
- Cache dan queue: Redis.

Untuk detail penuh, lihat dokumen di menu Dokumentasi.
