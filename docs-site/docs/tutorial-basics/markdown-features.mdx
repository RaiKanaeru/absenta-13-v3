# Konvensi API dan Data

Dokumen ini mencatat konvensi API agar data konsisten antara frontend dan backend.

## Format Waktu

- Gunakan format 24 jam `HH:MM` untuk jam pelajaran.
- Tanggal gunakan format `YYYY-MM-DD`.
- Timezone acuan adalah WIB.

## Status Absensi

Status standar yang dipakai:

- Hadir
- Izin
- Sakit
- Alpa
- Dispen

Pastikan nilai yang dikirim frontend sesuai dengan enumerasi di backend.

## Pola Response

Response yang direkomendasikan:

```
{
  "success": true,
  "data": [...],
  "message": "OK"
}
```

Untuk error:

```
{
  "success": false,
  "error": "Pesan error"
}
```

## Pagination

Jika endpoint memakai pagination, sertakan metadata seperti:

```
{
  "data": [...],
  "page": 1,
  "limit": 10,
  "total": 200,
  "totalPages": 20
}
```

## Contoh Request

```
GET /api/guru/student-attendance-history?page=1&limit=7
```

Konvensi ini memudahkan dashboard membuat ringkasan dan riwayat.
