# Template Excel untuk Export

Folder ini berisi file template Excel dari sekolah yang digunakan untuk export laporan.

## File yang Dibutuhkan

Copy file-file berikut ke folder ini:

1. `REKAP KETIDAKHADIRAN GURU 2025-2026.xlsx`
2. `REKAP KETIDAKHADIRAN KELAS X 2025-2026.xlsx`
3. `REKAP KETIDAKHADIRAN KELAS XI 2025-2026.xlsx`
4. `REKAP KETIDAKHADIRAN KELAS XII 2025-2026.xlsx`
5. `REKAP KETIDAKHADIRAN KELAS XIII 2025-2026.xlsx`
6. `PRESENSI SISWA 2025-2026 edit1.xlsx` (opsional)
7. `JADWAL PELAJARAN 2025-2026 (REVISI 2).xlsx` (opsional)

## Aturan Penting

- **JANGAN EDIT** file template setelah di-copy
- File ini adalah "golden template" - format asli sekolah
- Sistem akan load template, isi data, dan simpan sebagai file baru
- Rumus dan formatting akan tetap terjaga
